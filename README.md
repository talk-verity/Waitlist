Verity - Waitlist Landing Page

A sleek, modern, high-converting waitlist landing page for the Verity communications startup. Designed to capture early-access signups with a premium, tech-forward aesthetic.

Features

Modern Glassmorphism Design: Beautiful, translucent UI elements with dynamic, breathing background glows.
Fluid Animations: Complex CSS keyframes for a "liquid" background orb effect and continuous marquee feature carousels.
Interactive 3D Elements: Hover-responsive cards that gently tilt based on your cursor's precise position.
Magnetic Buttons: Micro-interactions that naturally draw the user's cursor toward the primary Call-To-Action.
Zero Dependencies: Built entirely with vanilla web technologies for instant loading, zero build-time, and maximum performance.
Fully Responsive: Optimized for a seamless experience across both desktop and mobile devices.
Tech Stack

HTML5: Semantic structure and form handling.
Vanilla CSS3: Advanced animations, CSS variables, mix-blend-mode, and Flexbox layouts.
Vanilla JavaScript: Real-time mathematical calculations for 3D tilt tracking, parallax background effects, and magnetic interactions.
